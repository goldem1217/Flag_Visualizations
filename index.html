Project Overview:


IN PROGRESS!

This full-stack project is just for practice - I'm brushing up on these languages and tools. When finished, this landing page will be an interactive dashboard created using JavaScript. Planned vizualizations include:

  • A flag color selector that will bring up images of flags based on the color selected.
  • A bar chart - showing number of flags of each color - that can be filtered by world region
  • Plots that show which flag colors fly over the greatest/least total area, number of people, and economic wealth
  • Infographics and trivia about color meanings and flag symbology
The goal (still to be realized) is a smooth, animated, interactive website.




This table contains the data that is used in the above visualizations. Use the dropdown to filter countries by the colors in their flags.
